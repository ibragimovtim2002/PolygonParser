Polygon Token API — демо

Уровень A

Описание: Получить баланс токена TBY у выбранного адреса.

Пример ввода: get_balance 0x51f1774249Fc2B0C2603542Ac6184Ae1d048351d

Пример ссылки: /api/balance/0x51f1774249Fc2B0C2603542Ac6184Ae1d048351d/

Уровень B

Описание: Получить баланс токена TBY нескольких адресов сразу.

Пример ввода: get_balance_batch ["0x51f177...", "0x4830AF..."]

Пример запроса: POST /api/balances/ с телом JSON:

{
  "addresses": [
    "0x51f1774249Fc2B0C2603542Ac6184Ae1d048351d",
    "0x4830AF4aB9cd9E381602aE50f71AE481a7727f7C"
  ]
}

Уровень C

Описание: Получить топ адресов по балансам токена TBY.

Пример ввода: get_top 5

Пример ссылки: /api/top/5/

Уровень D

Описание: Получить топ адресов по балансам токена TBY с датами последних транзакций.

Пример ввода: top_with_tx 5

Пример ссылки: /api/top_with_tx/5/

Уровень E

Описание: Произвольные операции с токеном (инфо, транзакции, холдеры и т.д.).

  • Информация о токене: /api/getinfo/token/0x1a9b54a3075119f1546c52ca0940551a6ce5d2d0/
  • Топ холдеров токена: /api/getinfo/holders/0x1a9b54a3075119f1546c52ca0940551a6ce5d2d0/5/
